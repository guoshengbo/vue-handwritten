# vue-hand-written

基于 Vue 2.x 的手写组件

<!-- ![screenshot](https://github.com/guoshengbo/vue-handwritten/blob/master/png.jpg?raw=true) -->

## 安装

```
npm i vue-hand-written --save
```

## 使用

```vue
<template>
  <div>
    <handwritten></handwritten>
  </div>
</template>

<script>
  import handwritten from 'vue-hand-written'

  export default {
      data () {
      }
      components: { handwritten }
  }
</script>
```
